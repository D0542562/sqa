========圈圈叉叉使用========

執行兩個 jar 檔之後

先點擊﹝oxSer.jar﹞的﹝wait for you﹞

點擊完成進入等待狀態

然後在﹝oxCli.jar﹞的 Your Address 打上開啟﹝oxSer.jar﹞的電腦 IP

如果是單機測試僅鍵入127.0.0.1即可

連線成功會跳出對話框提醒玩家已連線

接下來即可連線對戰圈圈叉叉

結束後想再玩必需重新開啟程式

ps:當 server 端進入等待狀態後僅有30秒連線時間

若30秒無人連線將會自動離線
